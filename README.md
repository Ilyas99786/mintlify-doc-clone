# Mintlify Documentation Website Clone (HTML + CSS)

This project recreates a desktop-first documentation-style website inspired by the Mintlify website.  
The main goal of this assignment is structural accuracy, layout readability, typography consistency, and section alignment using only HTML and CSS.

> Note: Original images/screenshots are not included in this repository.
